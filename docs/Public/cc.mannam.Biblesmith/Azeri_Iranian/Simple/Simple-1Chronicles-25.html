داوود و قوشون باشچيلاري، آسافين، هِيمانين و يِدوتونون نسلئندن بعضئلرئني خئدمت ائشئنه آييرديلار کي، طنبورلارلا، چنگلرله و سئنجلرله پيغمبرلئک اتسئنلر. بو خئدمت ائشئني گؤرَنلرئن سئياهيسي بودور: .1
آسافين نسلئندن: اونون اوغوللاري زَکّور، يوسئف، نِتَنيا، اَشه‌رِلا. آسافين باشچيليغي آلتيندا اولان اوغوللاري کي، پادشاهين امرئنه گؤره پيغمبرلئک ادئردئلر. .2
يِدوتونون نسلئندن: اونون اوغوللاري گِدَليا، صِري، اِشَعيا، شئمعي، خَشَبيا، مَتّئتيا. آتالاري يِدوتونون باشچيليغي آلتيندا اولان بو آلتي نفر ربّه شوکور و حمد اده‌رک چنگله پيغمبرلئک ادئردئلر. .3
هِيمانين نسلئندن: اونون اوغوللاري بوقّئيا، مَتّنيا، عوزّي‌اِل، شِبواِل، يِرئموت، خَنَنيا، خَناني، اِلئآتا، گئدّلتي، رومَمتي‌عِزِر، يوشبِقاشا، مَلّوتي، هوتئر و مَخه‌زي‌اوت. .4
بونلارين هاميسي پادشاهين گؤروجوسو هِيمانين اوغوللاري ائدئلر، چونکي هِيماني اوجالتماق باره‌ده تارينين وردئيي سؤزه گؤره، تاري اونا اون دؤرد اوغول و اوچ قيز ورمئشدي. .5
اونلارين هاميسي، ربّئن معبدئنده، آتالارينين باشچيليغي آلتيندا، تاري معبدئنئن خئدمتي اوچون سئنجلر، چنگلر و طنبورلار چاليب ائلاهي اوخويوردولار. لاکئن آساف، يِدوتون و هِيمان پادشاهين باشچيليغي آلتيندا ائدئلر. .6
ربّه ائلاهي اوخوماق اوچون اؤيرَدئلمئش آداملارين سايي، قوحوملاري ائله بئرلئکده کي، بونو باجارا بئلئردئلر، ائکي يوز هشتاد سکّئز نفر ائدي. .7
هاميسي، بؤيوکو ده، کئچئيي ده، موعَلّئمي ده، شاگئردي ده هَمن قايدا ائله وظئفه‌لري اوچون پوشک آتديلار. .8
بئرئنجي پوشک آسافين نسلئندن اولان يوسئفه دوشدو. اونلار اون ائکي نفر ائدئلر. ائکئنجئسي قوحوملاري و اوغوللاري ائله بئرلئکده گِدَليايا دوشدو. اونلار اون ائکي نفر ائدئلر. .9
اوچونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده زَکّورا دوشدو. اونلار اون ائکي نفر ائدئلر. .10
دؤردونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده يئصرئيه دوشدو. اونلار اون ائکي نفر ائدئلر. .11
بِشئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده نِتَنيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .12
آلتينجيسي اوغوللاري و قوحوملاري ائله بئرلئکده بوقّئيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .13
يِدّئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده يِسَراِلايا دوشدو. اونلار اون ائکي نفر ائدئلر. .14
سکّئزئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده اِشَعيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .15
دوقّوزونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده مَتّنيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .16
اونونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده شئمعئيه دوشدو. اونلار اون ائکي نفر ائدئلر. .17
اون بئرئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده عَزه‌رِله دوشدو. اونلار اون ائکي نفر ائدئلر. .18
اون ائکئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده خَشَبيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .19
اون اوچونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده شوبااِله دوشدو. اونلار اون ائکي نفر ائدئلر. .20
اون دؤردونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده مَتّئتيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .21
اون بِشئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده يِرِموتا دوشدو. اونلار اون ائکي نفر ائدئلر. .22
اون آلتينجيسي اوغوللاري و قوحوملاري ائله بئرلئکده خَنَنيايا دوشدو. اونلار اون ائکي نفر ائدئلر. .23
اون يِدّئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده يوشبِقاشايا دوشدو. اونلار اون ائکي نفر ائدئلر. .24
اون سکّئزئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده خَنانئيه دوشدو. اونلار اون ائکي نفر ائدئلر. .25
اون دوقّوزونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده مَلّوتئيه دوشدو. اونلار اون ائکي نفر ائدئلر. .26
ائيئرمئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده اِلياتايا دوشدو. اونلار اون ائکي نفر ائدئلر. .27
ائيئرمي بئرئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده هوتئره دوشدو. اونلار اون ائکي نفر ائدئلر. .28
ائيئرمي ائکئنجئسي اوغوللاري و قوحوملاري ائله بئرلئکده گئدّلتئيه دوشدو. اونلار اون ائکي نفر ائدئلر. .29
ائيئرمي اوچونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده مَخه‌زي‌اوتا دوشدو. اونلار اون ائکي نفر ائدئلر. .30
ائيئرمي دؤردونجوسو اوغوللاري و قوحوملاري ائله بئرلئکده رومَمتي عَزَره دوشدو. اونلار اون ائکي نفر ائدئلر. .31
