1. Tingtang hai, perkhuong hai darbenthek hai tuma chawibiek rongbawlnaa inhmang bik dingin Davida le a sipai kawmandarhai chun Asaf hai, Heman hai le Jeduthun hai naupahai chu an thlang suok a, chuonghai chu:
2. Asaf nauhaia inthokin Zakur, Josefa, Nethania le Asarela an nih. Lal inrawina hnuoia chawibiek rongbawlna nei hlak Asaf chu anni inrawitu a nih.
3. Jeduthun nauhai chu Gadalia, Zei, Jesaia, Simei, Hasabia le Matithia, an rengin paruk an nih. LALPA kuomah lawmthu hrila, tingtang leh an inpak huna chawibiek rongbawlna thuoitu an na, an pa Jeduthun chu anni inrawitu a nih.
4. Heman nauhai chu Bukia, Matania, Uziel, Subael le Jerimot; Hanania, Hanani, Eliatha, Gidalti le Romamti-Ezer; Josbekasa, Malothi, Hothir le Mahaziot an nih.
5. Hienghai po po hi lal zawlnei Heman naupahai an nih. Ama chu Pathienin chawisang dinga a tiem a ni leiin naupa sawmpali le naunu pathum a pek.
6. An reng chun an pa rawiin darbenthek le perkhuong le tingtang tuma hla le rimawi remin Pathien inah rong an bawl a. Asaf hai, Jeduthun hai le Heman hai ruok chu lalin a ruot an nih.
7. An tam tieng chu, an unauhai leh hla sak le rimawi tum inchuk lai mek le a thiem ta sa po chu, zahni le sawmriet le pariet an nih.
8. A chin a lien, inchuk lai le zirtirtu khom ni hai sien, an sinthaw ding seng enna dingin thum an vawr a.
9. Thumvawr hmasa tak chu Asaf naupa Josefa chungah a tlu a, ama le a naupahai le a sunghai chu sawm le pahnih an nih. A voi hnina chu Gedalia le a unauhai le naupahai sawm le pahnihai chungah;
10. a voi thumna chu Zakur le a unauhai le naupahai sawm le pahnihai chungah;
11. a voi lina chu Izri (Zeri) le a unauhai le naupahai sawm le pahnihai chungah;
12. a voi ngana chu Nethania le a unauhai le a naupahai sawm le pahnihai chungah;
13. a voi rukna chu Bukia le a unauhai le naupahai sawm le pahnihai chungah;
14. a voi sarina chu Jesarela le a unauhai le a naupahai sawm le pahnihai chungah;
15. a voi rietna chu Jesaia le a unauhai le naupahai sawm le pahnihai chungah;
16. a voi kuona chu Matania le a unauhai le a naupahai sawm le pahnihai chungah;
17. a voi sawmna chu Simei le a unauhai le naupahai sawm le pahnihai chungah;
18. a voi sawm le pakhatnaa chun Azarel le a unauhai le nauhai sawm le pahnihai chungah;
19. a voi sawm le voi hninaa chun Hashabia le a unauhai le nauhai sawm le pahnihai chungah;
20. a voi sawm le voi thumnaa chun Subael le a unauhai le naupahai sawm le pahnihai chungah;
21. a voi sawm le voi linaa chun Mathithia le a unauhai le naupahai sawm le pahnihai chungah;
22. a voi sawm le voi nganaa chun Jerimot le a unauhai le naupahai sawm le pahnihai chungah;
23. a voi sawm le voi ruknaa chun Hanania le a unauhai le naupahai sawm le pahnihai chungah;
24. a voi sawm le voi sarinaa chun Josbekasa le a unauhai le naupahai sawm le pahnihai chungah;
25. a voi sawm le voi rietnaa chun Hanani le a unauhai le naupahai sawm le pahnihai chungah;
26. a voi sawm le voi kuonaa chun Malothi le a unauhai le naupahai sawm le pahnihai chungah;
27. a voi sawmhninaa chun Eliatha le a unauhai le naupahai sawm le pahnihai chungah;
28. a voi sawmhni le voi khatnaa chun Hothir le a unauhai le naupahai sawm le pahnihai chungah;
29. a voi sawmhni le voi hninaa chun Gidalti le a unauhai le naupahai sawm le pahnihai chungah;
30. a voi sawmhni le voi thumnaa chun Mahaziot le a unhai le naupahai sawm le pahnihai chungah;
31. a voi sawmhni le voi linaa chun Romamti-Ezer le a unauhai le naupahai sawm le pahnihai chungah a tlu a.
