1. Mani suoi nama thuthlung namnghettuhai chu: Nehemia, ram optu, Hakalia naupa; thiempuhai laia inthokin Zedekia,
2. Seraia, Azaria, Jeremia,
3. Pasur, Amaria, Malkija,
4. Hatus, Sebania, Maluk,
5. Harim, Meremot, Obadia,
6. Daniel, Ginethon, Baruk,
7. Mesulam, Abija, Mijamin,
8. Maazia, Bilgai le Semaia, thiempu vong an nih.
9. Levi chia inthoka namnghettuhai chu Azania naupa Jesua, Henedad sungkuoa mi Binui, le Kadmiel
10. le an Levi chanpui Sebania, Hodia, Kelita, Pelaia, Hanan,
11. Mika, Rehob, Hasabia,
12. Zakur, Serebia, Sebania,
13. Hodia, Bani le Beninu hai an nih.

15. Buni, Azgad, Bebai,
16. Adonija, Bigvai, Adin,
17. Ater, Hezekia, Azur,
18. Hodia, Hasum, Bezai,
19. Harif, Anathot, Nebai,
20. Magpias, Mesulam, Hezir
21. Mesezabel, Zadok, Jadua,
22. Pelatia, Hanan, Anaia,
23. Hosea, Hanania, Hasub,
24. Halohes, Pilha, Sobek,
25. Rehum, Hasabna, Maaseia,
26. Ahia, Hanan, Anan,
27. Maluk, Harim le Baana hai an nih.
28. Namnghettu mi danghai chu thiempuhai, Levihai, kotkhar vengtuhai, hla saktuhai, tempula siehlawhai le Pathien dan zawm tuma an se vela umhai laka insie hranghai le an nuhmeihai le chinchang hre thei tah an naupahai le naunuhai chu an nih.
29. Hienghai hin an unauhai le an mi lienhai thu chu chel tlatin, Pathien siehlaw Mosie pek Pathien dan zawm a, LALPA, kan Pathien thupek po po le a rorelnahai le thuruothai chu fimkhur taka zawm le vong dingin inkham meuin thu kan thlung a.
30. "Kan kawl le kienga chenghai chu kan naunuhai inneitir loa, kan naupahai ta ding khoma an naunuhai chu nei bok lo dinga inkhamin thu kan intiem.
31. "Kan kawl le kienga mihaiin thuomhnaw amanih fak ding thil amanih zor dinga Sabat ni le umni khama ni kan inser changa an hung phur hrim hrim chu inchawpek lo dingin kan intiem. Chun, kum sarina zatin lo thaw loa ram chu inawl ding le kan laka Juda mi leibat chu thai bo dingin inkhamin thu kan intiem bok.
32. "LALPA in siemna dinga sum mamaw tawk a um theina dingin, kum tin shekel hmun thuma thea hmun khat pek dinga thupek um chu sukpuitling dingin mawphurna lak kan intiem.
33. Chu thawlawm chun chatlak loa hlanbei sie ding dam, bu le bal inhlanna le pumraw thilhlana ding dam, Sabat ni le thla thar lawmna le kum tina kut hrang hrang hmangnaa ding dam, thil thienghlim inhlannaa ding dam, Israel mihai ta dinga inremna siemna dinga suol thawinaa ding dam le kan Pathien tempula thaw tul dang po poa hmang ding mamawhai chu phuhruk a tih.
34. "Danin a phut anga LALPA kan Pathien maichama ti ding kum tina a hun bi ruot ang thlapa kan Pathien tempula thing hung latu ding sungkuohai hrietna dingin thiempuhai, Levihai le mipuihaia inthokin thum vawrin kan sukthluk.
35. "Kan lo tharsuok bu le bal le thlai thar hmasa tak le kan thing le thei kung ra hmasa takhai chu LALPA tempula hung chawi rop dingin kan intiem.
36. Kan naupa pieng hmasa takhai le kan ran pieng hmasa tak, kan bawng ruol le beram ruolhai laia pieng hmasa takhai chu danin a phut anga LALPA ta dinga kan Pathien tempula rongbawltu thiempu kuoma hung inhlan dingin kan intiem.
37. "Kan bei hmet hmasa tak hai, kan bu le bal thilhlan hai, thing le thei chi tinreng ra hmasa tak hai, uoiin thar le hriek thar hmasa tak hai laia a tha tak pei chu LALPA ina a siekhawmnaah thiempu kuomah hung chawi dingin kan intiem. Kan lo tharsuok sawma pakhat changtu ding le chu chu kan umna khaw tina la khawmtu dinga ruot Levihai kuoma chun kan lo tharsuok sawma pakhat hung chawi seng dingin kan intiem.
38. Levi chihaiin sawma pakhat an laka inthoka sawma pakhat chu Aron thlaa mi thiempuhai ta dingin kan Pathien tempula ro siekhawmnaah an pek ve hlak ding a nih.
39. Israel mihai le Levi chihaiin bu le bal thilhlan dam, uoiin thar dam le oliv hriek thar dam an inhlanhai chu tempulah hung chawi an ta, rongbawltu thiempu le kotkhar vengtu le hla saktuhai umna bula bur le bel siekhawmna kudama sie dingin an hung chawi seng ding a nih."Kan Pathien in hi ngaitha lo dingin kan rengin kan intiem" tiin.
