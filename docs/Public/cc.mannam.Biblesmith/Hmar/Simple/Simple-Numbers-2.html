1. LALPAN Mosie le Aron chu a biek a:
2. "Israel thlahai chun Intuokna Puon ina inthoka iemani chena hlaah mani sungkuo inchikna puonzar seng tarin, an pawl thaw dan ang sengin puon in an kei ding a nih.
3. "Sak tieng, nisa suokna tieng chun Juda pawlin an pahnam inchikna seng tarin puon in kei an ta, an thuoitu chu Aminadab naupa Nason ning a ta
4. an tam zat chu singsari le sangli le zaruk a nih.
5. Anni dotah Isakar pahnamin puon in kei an ta, an thuoitu chu Zuar naupa Nethanel ning a ta,
6. an tam zat chu singnga le sangli le zali a nih.
7. A dotah Zebulun pahnam ning an ta, an thuoitu chu Helon naupa Eliab ning a ta,
8. an tam zat chu singnga le sangsari le zali a nih.
9. Juda pahnam inrawina hnuoia um po chu, pawl the dan dungzuiin, nuoikhat le singriet le sangruk le zali an nih. Anni hin hma thuoi an tih.
10. "Sim tieng pangah mani pahnam inchikna seng tarin Reuben pawlhai chu um an ta, an thuoitu chu Sedeur naupa Elizur ning a ta,
11. an tam zat chu singli le sangruk le zanga a nih.
12. A dotah Simeon pahnam um an ta, an thuoitu chu Zurisadai naupa Selumiel ning a ta,
13. an tam zat chu singnga le sangkuo le zathum a nih.
14. Anni dotah Gad pahnam um an ta, an thuoitu chu Deuel naupa Eliasaf ning a ta,
15. an tam zat chu singli le sangnga le zaruk le sawmnga a nih.
16. Reuben pahnam inrawina hnuoia um po chu an pawl the dan dungzuiin, nuoi khat le singnga le sangkhat le zali le sawmnga an nih. Anni po hi pawl hninaa suok ding an nih.
17. "Chu zoah Intuokna Puon in le Levi pawlhai chu a laiah thangin suok an ta, riebuka an inkul dan indot ang peiin, mani pawl seng inchikna puonzar khaiin, suok an tih.
18. "Thlang tieng panga chun mani pahnam inchikna seng tarin Efraim pawlhai chu um an ta, an thuoitu chu Amihud naupa Elisama ning a ta,
19. an tam zat chu singli le zanga a nih.
20. Anni dotah Manase pahnam um an ta, an thuoitu chu Pedazur naupa Gamaliel ning a ta,
21. an tam zat chu singthum le sanghni le zahni a nih.
22. A dotah Benjamin pahnam um an ta, an thuoitu chu Gideoni naupa Abidan ning a ta,
23. an tam zat chu singthum le sangnga le zali a nih.
24. Efraim pahnam inrawina hnuoia um po chu, an pawl the dan dungzuiin, nuoikhat le sangriet le zakhat an nih. Anni pawl thumnaa suok dinghai chu an nih.
25. "Hmar tieng panga chun Dan pawla mihai anni inchikna seng tarin um an tih. Dan pawl thuoitu chu Amisadai naupa Ahiezer ning a ta,
26. an tam zat chu singruk le sanghni le zasari a nih.
27. A dotah Aser pahnam um an ta, an thuoitu chu Okran naupa Pagiel ning a ta,
28. an tam zat chu singli le sangkhat le zanga a nih.
29. Anni dotah Naftali pahnam um an ta, an thuoitu chu Enan naupa Ahira ning a ta,
30. an tam zat chu singnga le sangthum le zali a nih.
31. Dan pahnam inrawina hnuoia um po chu, an pawl the dan dungzuiin, nuoi khat le singnga le sangsari le zaruk a nih. Anni hi mani pahnam puonzar seng khaia a nuhnung taka suok ding an nih."
32. Riebuka um Israel thlahai, pahnam tina mi an tiem zat chu nuoi ruk le sangthum le zanga le sawmnga a nih.
33. LALPAN Mosie thu a pek ang khan, Levi chia mihai chu Israel thlahai laia tiem sa an ni ve nawh.
34. Chuong ang chun Israel thlahai chun an thaw a, LALPAN Mosie thu a pek ang peiin, mani pawl inchikna puonzar tarna sengah pawl tinin puon in an kei a, chuong ang chun a chi chiin, a sungkuo kuoin an intawl suok hlak a nih.
