Numbers (1/36)  →
1. Aigupta rama inthoka Israelhai an suok hnung a kum hnina le thla hnina ni hmasa taka chun Sinai Thlalera Intuokna Puon ina chun LALPAN Mosie a biek a.
2. "Israelhai po po chu an chi hnam, an sungkuo le an hming indot dan ang peiin tiem rawh.
3. Israel mi po po, kum sawmhnia inthoka a chung tieng, sipaia thang thei pasal po chu nang le Aronin a pawl pawlin tiem vong ro.
4. Pahnam tina sungkuo lukhaitu peiin an thangpui ding che a nih.
5. A thangpuituhai ding che hming chu: Reuben hnama inthokin Sedeur naupa Elizur;
6. Simeon hnama inthokin Zurishadai naupa Selumiel;
7. Juda hnama inthokin Aminadab naupa Nashon;
8. Isakar hnama inthoin Zuar naupa Nethanel;
9. Zebulun hnama inthokin Helon naupa Eliab;
10. Josefa thla, Efraim hnama inthokin Amihud naupa Elisama le Manase hnama inthokin Pedazur naupa Gamaliel;
11. Benjamin hnama inthokin Giddeoni naupa Abidan;
12. Dan hnama inthokin Amisadai naupa Ahiezer;
13. Aser hnama inthokin Okran naupa Pagiel;
14. Gad hnama inthokin Deuel naupa Eliasaf;
15. Naftali hnama inthokin Enan naupa Ahira ning an tih.
16. Hienghai hi mipuihai laia inthoka thlang, an pi le puhai hnam thuoituhai, Israel pahnam tina an lukhaituhai chu an nih.
17. Hieng an hming hril tahai hi Mosie le Aron chun an thuoi a,
18. a thla hnina ni khat ni chun mipui an ko khawm vong a, a pahnam hnamin, a sungkuo kuoin, an hming indot dan ang peiin kum sawmhnia inthoka a chung tieng po hming chu an ziek lut a,
19. LALPAN Mosie kuoma thu a pek angin Sinai thlalera chun an tiem vong a.
20. Israel nau lutir Reuben thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot ang peiin an ziek lut a,
21. Reuben hnama mi chu singli le sangruk le zanga an nih.
22. Simeon thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
23. Simeon hnama mi po chu singnga le sangkuo le zathum an nih.
24. Gad thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
25. Gad hnama mi po chu singli le sangnga le zaruk le sawmnga an nih.
26. Juda thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
27. Juda hnama mi po chu singsari le sangli le zaruk an nih.
28. Isakar thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
29. Isakar hnama mi po chu singnga le sangli le zali an nih.
30. Zebulun thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
31. Zebulun hnama mi po chu singnga le sangsari le zali an nih.
32. Josefa nauhaia inthokin: Efraim thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
33. Efraim hnama mi po chu singli le zanga an nih.
34. Manase thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
35. Manase hnama mi po chu singthum le sanghni le zahnih an nih.
36. Benjamin thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
37. Benjamin hnama mi po chu singthum le sangnga le zali an nih.
38. Dan thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
39. Dan hnama mi po chu singruk le sanghni le zasarih an nih.
40. Aser thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
41. Aser hnama mi po chu singli le sangkhat le zanga an nih.
42. Naftali thlaa inthoka pasal kum sawmhni le a chung tieng, raldo thei chin hai chu an pahnam le sungkuo rikawta an inchik dan angin, an hming indot peiin an ziek lut a,
43. Naftali hnama mi po chu singnga le sangthum le zali an nih.
44. Chuonghai po chu an nih Mosie le Aron le Israel thlahai pahnam tin thuoitu lu sawmpahnihaiin an tiem chu.
45. Israel thla sungkuo tina mi, raldo thei pasal, kum sawmhni le a chung tieng, Israel sipaia thang thei po chu an tiem vong a,
46. an rengin nuoiruk le sangthum le zanga le sawmnga an nih.
47. Levi hnama sungkuohai ruok chu tiem sa an ni ve nawh.
48. LALPAN Mosie kuomah,
49. "Levi hnam chu Israel pahnamhai lai tiem sa ve naw rawh.
50. Levi chihai chu Thuthlung biekbuk le a sunga hmangruo le thil um po po enkoltu dingin ruot tawl lem rawh. Biekbuk le a hmangruo po po zawn hlak an ta, enkol an ta, biekbuk chu um nghang an tih.
51. An inson ding zatin biekbuk chu Levihai chun thiek an ta, an zar nawk ding pha leh Levihai bokin zar nawk an ta, mi dang hung inrol taphot chu sukhlum ning an tih.
52. Chun, Israel thlahai chun mani hnam puonzar inkhaina hmun sengah, an tam dan ang peiin, a pawl pawlin puon in kei an tih.
53. Israelhai chungah lungsenna a tlak nawna dingin Levihai chun Thuthlungna biekbuk huol velin puon in kei an ta, an veng ding a nih" tiin a lo hril ta si a.
54. Israelhai chun LALPAN Mosie kuoma a hril ang tak chun an thaw a.
