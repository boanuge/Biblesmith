1. Isakar naupahai chu Tola, Pua, Jasub le Simron, an rengin pali an nih.
2. Tola naupahai chu Uzi, Refaia, Jeriel, Jamai, Ibsam le Samuel an na, chuonghai chu pahnam tin lukhaituhai an nih. Davida inlal laia Tola thlahai laia indo mi hming an inchik po chu mi singhni le sanghni le zaruk an nih.
3. Chun, Uzi naupa chu Izrahia a na, Izrahia naupahai chu Mikael, Obadia, Joel le Isia an na, an rengin pahnam lukhaitua thang vong an nih.
4. Nuhmei tam tak le nau tam tak an nei leiin an sungkuo intiemnaa indoa thang thei chin po chu mi singthum le sangruk an nih.
5. Chun, Isakar hnam pumpui intiemnaa indoa ramtang thei chin an tiem po chu singriet le sangsari an nih.
6. Benjamin naupahai chu Bela, Beker le Jediael an nih.
7. Bela naupahai chu Ezbon, Uzi, Uziel, Jerimot le Iri, an rengin panga an na, sungkuo lukhaitu vong an nih. An thlahai tiemnaa indoa ramtang thei an tiem po chu singhni le sanghni le sawmthum an nih.
8. Beker naupahai chu Zemira, Joash, Eliezer, Elioenai, Omri, Jeremot, Abija, Anathot le Alemet an nih. An rengin Beker naupahai vong an nih.
9. An thlahai tiemnaa sungkuo lukhaituhai an nih. An renga ral doa ramtang thei mi an tiem po chu singhni le zahnih an nih.
10. Jediael naupa chu Bilhan a na, Bilhan naupahai chu Leush, Benjamin, Ehud, Kenana, Zethan, Tarsis le Ahisahar an nih.
11. Hieng Jediael thlahai hi sungkuoa lukhaitu vong an nih. An sungkuoa ral doa ramtang thei zat chu singkhat le sangsari le zahni an nih.
12. Anni hnam laia Supim le Hupim hai chu Ira thlahai an na, Husimhai chu Aher thlahai an nih.
13. Naftalia naupahai chu Jaziel, Guni, Jezer le Sillum (Salum) an na, chuonghai chu Jakob hmei Bilha thlahai an nih.
14. Manase thlahai chu: Asriel chu a hmei Aram mi thlaa mi a nih. Hi hmei hin Gilead pa Makir a hring a,
15. Makir chun Hupim le Supim mi nuhmeiah a nei a. A farnu hming chu Maaka a nih. A thla dang chu Zelofehad a na, ama chun nau nuhmei chauh a nei.
16. Makir nuhmei Maaka chun nau pasal a hming Peres a nei a. A unaupa hming chu Seres a na, a naupahai chu Ulam le Rakem an nih.
17. Ulam naupa chu Bedan a nih. Chuonghai chu Manase thla Makir naupa Gilead naupahai chu an nih.
18. Chun, a farnu Hamoleket chun Ishod, Abiezer le Mala a hring a.
19. Semida naupahai chu Ahian, Sekem, Likhi le Aniam an nih.
20. Efraim naupa chu Suthela a na, ama naupa chu Bered a na, chu naupa chu Tahat a na, chu naupa chu Eleada a na, chu naupa chu Tahat a na,
21. chu naupa chu Zahad a na, chu naupa chu Suthela a nih. A nau danghai Ezer le Elead chu Gath khuoa mihai ran ruol inruk an tumnaah a ram mihaiin an lo that a.
22. An pa Efraim chun ni tam tak a sun a, a unauhai chun an hung ral a.
23. Chun, a nuhmei chu a pawl a, nau a pai a, nau pasal a hring a, a insung vangduoinan a tlak buok lei chun a hmingah Beria a sak a.
24. A naunu hming chu Seera a na, ama chu Beth Horon khuo chuonchung le chuonhnuoi le Uzen Seera khuohai indintu a nih.
25. Efraim chun naupa dang a hming Refa a nei a, chu naupa chu Resef a na, chu naupa chu Tela a na, chu naupa chu Tahan a na,
26. chu naupa chu Ladan a na, chu naupa chu Amihud a na, chu naupa chu Elisama a na,
27. chu naupa chu Nuna a na, chu naupa chu Josua a nih.
28. An chanpuol ram le umhmun an khuornahai chu Bethel le a khawperhai, sak tieng Naaran, thlang tieng Gezer le a khawperhai, Sekem le a khawperhai le Ayya le a khawperhai chenin.
29. Manase-hai ramri bula khuohai chu Beth San, Tamanak, Megido le Dor le an khawperhai an nih. Israel naupa Josefa thlahai chu hieng khuoa hai hin an um a nih.
30. Aser naupahai chu Imna, Isva, Isvi le Beria an na, an farnu chu Sera a nih.
31. Beria naupasahi chu Heber le Malkiel an na, Malkiel naupa chu Birzait a nih.
32. Heber naupahai chu Jaflet, Somer le Hotham an na, an farnu chu Sua a nih.
33. Jaflet naupahai chu Pasak, Bimhal le Asvat an nih.
34. Somer naupahai chu Ahi, Roga, Huba le Aram an nih.
35. A unaupa Helem naupahai chu Zofa, Imna, Seles le Amal an nih.
36. Zofa naupahai chu Sua, Harnefer, Sual, Beri, Imra,
37. Bezer, Hoda, Sama, Silsa, Ithran (Jether) le Beera an nih.
38. Jether naupahai chu Jefune, Pispa le Ara an nih.
39. Ula naupahai chu Ara, Haniel le Rizia an nih.
40. Hieng Aser thlahai hi pahnam tina lukhaituhai, thlang fal, mi huoisen le thuoitu challanghai chu an nih. An thlahai, ral doa ramtang thei an tiemhai po chu singhni le sangruk an nih.
