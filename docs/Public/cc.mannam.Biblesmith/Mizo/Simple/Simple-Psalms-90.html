1. Lalpa, nang chu chhuan zawng zawng chenah Kan awmna i ni thin.
2. Tlangte pian hmaa, Leilung leh khawvel i din hma pawh khan, Chatuan ata chatuan thleng pawhin nang chu Pathian i ni.
3. Nang chuan mihring hi boralnaah i kirtir thin a; "Mihring fate u, kir leh rawh u,m" i ti thin.
4. Nang ngaih chuan kum sangkhat hi Nimin ral ta ang lek kha ani si a, Zan ven khat chhung ang lek bawk a ni.
5. Anni chu tui lian angin i len bo thin a; thang khat mu ang chauh an ni; Zingah chuan hlobet chawr chuak ang an ni.
6. Zingah chuan a lo duahin a chawr chhuak a; Tlai lamah chuan ah thlukin a awm a, a vuai thin.
7. I thinurnaa tihboralin kan awm a, I thinrimnaa tihmanganin kan awm bawk si a.
8. Kan khawlohnate chu i hmaah i dah a, Kan sual zep rukte chu i hmel engah i dah ta a.
9. Kan dam chhung ni zawng zawng hi i thinrimnaah a ral thin a: Kan dam chhung kumte hi thawnthu angin kan titawp thin.
10. Kan dam chhung kumte chu kum sawmsarih ani a, Chakna avangin kum sawmriat pawh a ni thei a; Nimahsela a reina chu thawhrimna leh lungngaihna mai a ni; A ral thuai a, keini pawh kan thlawk bo thin a nih hi.
11. Tuing nge i thinurna thiltihtheihzia hria a, Nangmah tih turzia ang i thinrimna chu hria le?
12. Kan dam chhung nite chhiar dan tur min zirtir ang che, Finna thinlung kan neih phah hialna turin.
13. Aw LALPA, lo kir leh ang hche; eng chen nge i thinur dawn? I chhiahhlawhte chung thu-ah chuan i rilru lamlet ang che.
14. Aw, zingah chuan i ngilneihnain mi titlai ang che; Kan dam chhung zawng zawng hian kan hlima kan lawm theihna turin.
15. Nangin min tihhreawm ni leh, thil tha lo kan tawh kum chhung hun angte khan min tilawm ang che.
16. I hnathawh chu i chhiahhlawhte hnenah lang se la, I ropuina chu an fate hnenah lang rawh se.Tin, LALPA kan Pathian mawina chu kan chungah awm rawh se: Kan kut hnathawh hi kan chungah tinghet bawk ang che; A ni, kan kut hnathawh hi tinghet ang che.
17. Tin, LALPA kan Pathian mawina chu kan chungah awm rawh se: Kan kut hnathawh hi kan chungah tinghet bawk ang che; A ni, kan kut hnathawh hi tinghet ang che.
