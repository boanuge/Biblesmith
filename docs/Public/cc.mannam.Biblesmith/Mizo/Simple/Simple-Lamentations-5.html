1. Aw LALPA, kan chunga lo thleng chu hre reng ang che; Ngai la, kan mualphona hi en teh!
2. Kan roluahte chu hnamdang mite hnena pk a ni zo ta a, Kan inte chu hmelhriatloh mite hnenah.
3. Fahrahte leh pa neilote kan ni ta a; Kan nute hmeithai ang mai an ni ta.
4. Kan tuisikte man neiin kan in ta a, Kan thingtuahte pawh kan hnenah hralh a ni ta.
5. Nghawnga manin kan awm a; kan chau va chawlh reng kan nei lo.
6. Aigupta leh Assuria hnenah kut kan dawh a; Kham khawp chaw hmu turin.
7. Kan pate zawng an sual a, an awm ta lo a; An khawlohnate kan phur a ni.
8. Bawihte chu kan chungah an lal a; An kut ata min chhanchhuak tur tumah an awm lo.
9. Nun chan hlauhawm khawpin kan chaw kan ei th n; Thlalera khandaih vang chuan.
10. Kan vunte chu thuk ang maiin a sa a; tam kang sa vangin.
11. Zion-ah hmeichhiate chu an pawng sual a, nulate chu Juda khawpuiah te.
12. Lal fate an kutin an khai a, upate hmel an zah lo.
13. Tlangvalte buh her khawl an hert r a; Naupangte thingtuah tel hnuaiah an pai chawn chawn a.
14. Upaten kulh kawngka an kalsan a, tlangvalten an hlasak.
15. Kan thinlung hlimna a chuai ta; kan lam lusunah a chang ta.
16. Kan lu ata lallukhum a tla ta; Kan chung a pik e, kan tisual ta si.
17. Hemi avang hian kan thinlung a chau a, Heng avang hia kan mit a thim.
18. Zion tlang chu a tiau tak avangin, Sihalte a chhungah an leng.
19. Nang chu LALPA, kumkhuain i lal a; i lalthutthleng chu chhuan zawng zawng thlengin.
20. Eng vangin nge kumkhuaa min theihnghilh a, chu leh chen min kalsan le?
21. Aw LALPA, nagin i lam mi hawit r la, tichuan kan lo hawi ang; hmasang angin kan nite tithar ang che.A nih leh, nangin mi paih thla der em ni? Kan chungah i thin a ur nasa hle em ni?
22. A nih leh, nangin mi paih thla der em ni? Kan chungah i thin a ur nasa hle em ni?
