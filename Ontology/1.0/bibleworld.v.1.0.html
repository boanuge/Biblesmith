Description - bibleworld v1.0

This is the ontology to describe things in the bible.

  • This owl is focused on the genealogy of Jesus Christ.

    • Based on: Genesis 5:1-32
    • Based on: Matthew 1:1-17

  • Available Classes

    • Class: EarthlyWorld, Living, Human, Man
    • ObjectProperty: hasFamily, hasAncestor, hasParent, hasFather, hasOffspring, hasChild, hasSon
