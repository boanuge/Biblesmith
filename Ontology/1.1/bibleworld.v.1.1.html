Description - bibleworld v1.1

This is the ontology to describe things in the bible.

  • This owl is focused on the genealogy of Jesus Christ.

    • Based on: Genesis, Matthew

  • Available Classes

    • Class: EarthlyWorld, Living, Human, Man, Woman
    • ObjectProperty: hasFamily, hasAncestor, hasParent, hasFather, hasMother, hasOffspring, hasChild, hasDaughter, hasSon, hasSibling, hasBrother, hasSister, hasSpouse, hasHusband, hasWife, hasChangedNameTo
    • DatatypeProperty: isDeadAtTheAge, isBornAtTheFatherAge, hasGoneToSpiritualWorldAtTheAge, hasChangedNameAtTheAge

  • This version has been updated from v1.0 with more details about the ages, sibling and spouse.
